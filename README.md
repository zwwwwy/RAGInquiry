# RAGInquiry
### 简介  
针对券商投行招股说明书以及审核问询函撰写等业务场景的RAG智能问答系统，支持模板式回复生成、按语义检索并核对数据、多类型材料交叉检索、向量库文档管理及对话历史管理的功能。  
### 核心架构
基于LangChain搭建核心RAG流程，采用Chroma向量库实现语义检索，使用Qwen3-max结合上下文的内容与参考资料生成模板化的合规回答或其他定制化回答，并通过SQLite管理会话历史。  
### 优化策略
使用基于通义text-embedding-v4模型完成文档语义切分与向量入库，设计向量+关键词多路召回策略，结合Rerank重排算法，选取余弦相似度最优的指定数量文档片段按文档类型供模型调用，提升检索准确性与回答匹配度。  
### 文档分类
支持txt/pdf/excel等多格式文件解析，通过元数据实现可灵活扩展的自定义文件类型，可根据用户实际需求自行交叉查找并引用多种类型的文件，相互论证或指出潜在的错误。
